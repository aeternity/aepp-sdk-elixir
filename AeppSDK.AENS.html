    aepp_sdk_elixir v0.5.3 AeppSDK.AENS

    Module for Aeternity Naming System, see: https://github.com/aeternity/protocol/blob/master/AENS.md. Contains all naming-related functionality.

    In order for its functions to be used, a client must be defined first. Client example can be found at: AeppSDK.Client.new/4.
