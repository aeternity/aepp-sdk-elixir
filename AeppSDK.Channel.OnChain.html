    aepp_sdk_elixir v0.5.3 AeppSDK.Channel.OnChain

    Module for Aeternity On-Chain channel system, see: https://github.com/aeternity/protocol/blob/master/channels/ON-CHAIN.md Contains On-Chain channel-related functionality.

    In order for its functions to be used, a client must be defined first. Client example can be found at: AeppSDK.Client.new/4.
