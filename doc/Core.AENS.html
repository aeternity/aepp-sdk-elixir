    aepp_sdk_elixir v0.1.0 Core.AENS

    Module for Aeternity Naming System, see: https://github.com/aeternity/protocol/blob/master/AENS.md. Contains all naming-related functionality.

    In order for its functions to be used, a client must be defined first. Client example can be found at: Core.Client.new/4.
